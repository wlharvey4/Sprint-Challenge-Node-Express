# Assessing Note - Express Fu

## Answers to Questions

1. *What is Node?* Server-side engine for running JavaScript outside of
   the browser. 
   
2. *What is Express?* A Node package that makes creating Node servers a
   piece of cake.
   
3. *What is Middleware?* Additional Node packages and pieces of Express
   that manage requests and responses in between receipt and send.
   
4. **What is an endpoint?* A URL that a server responds to when requests
   come into the server using that URL.
